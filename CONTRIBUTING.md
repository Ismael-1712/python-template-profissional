# Guia de Contribuição

Ficamos felizes com seu interesse em contribuir!

## Reportando Bugs

Por favor, abra uma "Issue" detalhando o bug, o que você esperava que acontecesse e como podemos reproduzi-lo.

## Sugerindo Melhorias

Abra uma "Issue" descrevendo sua ideia e por que ela seria benéfica para o projeto.

## Pull Requests

1. Faça um "Fork" do repositório.
2. Crie um novo *branch* para sua feature (`git checkout -b feature/minha-feature`).
3. Faça o *commit* das suas mudanças (`git commit -m 'feat: Adiciona minha feature'`).
4. Faça o *push* para o seu *branch* (`git push origin feature/minha-feature`).
5. Abra um "Pull Request".
