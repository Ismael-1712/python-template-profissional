# Política de Segurança

## Reportando uma Vulnerabilidade

Por favor, **não** abra uma "Issue" pública para vulnerabilidades de segurança.

Envie um e-mail diretamente para `seu-email@dominio.com` (o mesmo do `pyproject.toml`) com o título "VULNERABILIDADE DE SEGURANÇA" e nós responderemos o mais rápido possível.
